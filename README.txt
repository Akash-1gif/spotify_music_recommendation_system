the following libraries has to be installed before executing the program:
numpy
pandas
matplotlib
seaborn
sklearn
joblib
imblearn

Initially start with data_preprocessing1.ipynb, go to kernel and select restart and run all cells. This file consists of entire data cleaning procedure (Step 1: Data Preprocessing)

After running the data_preprocessing.ipynb, run cluster_analysis_and_model_evaluation2.ipynb. Adjust the number of clusters count after generating silhouette scoring graph. This file consists of clustering of data, model training and evaluation. (Step 2 to Step 5 - Cluster Analysis, Training and Testing Data, Balancing the Training Dataset and Model Evaluation)

At last run the songs_recommendation3.ipynb file. Follow the same steps to generate recommendations for different songs and group of songs (Step 6 - SOngs Recommendation)
